# OracleDatabase
Repositório dos códigos **SQL** e **PL/SQL** dos bancos de dados Oracle

## Diretórios
### Config
Contém os arquivos de configuração necessários para o deploy e a reversão via *Jenkins*

### Deploy
Contém os códigos **SQL** e **PL/SQL** para publicação no banco de dados

### Revert
Contém os códigos **SQL** e **PL/SQL** para reversão da publicação em caso de erros
